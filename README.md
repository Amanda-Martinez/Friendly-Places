# Friendly-Places
Trouver un endroit sympa pour faire des trucs.

## Pour insérer des exemples de nouveaux lieux

Dans la console :
```javascript
> generatePlaces()
```

## Pour insérer un nouveau lieu

Dans la console :

``` javascript
> world.new({name: 'Fairview', city: 'Montpellier', location: {lt: '43.6117398', lg: '3.8767571'}})
```
